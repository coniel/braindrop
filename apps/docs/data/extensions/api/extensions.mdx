# Extensions

<Description>
  An interface for managing MindDrop's extensions.
</Description>

### getExtension

Returns a registered extension by ID. Throws an `ExtensionNotRegisteredError` if the extension is not registered.

```ts
Extensions.get(extensionId: string): Extension
```

<ArgsTable
  data={[
    {
      name: 'extensionId',
      type: 'string',
      required: true,
      description: 'The ID of the extension to retrieve.',
    },
  ]}
/>

### getEnabled

Returns an array of all enabled extensions.

```ts
Extensions.getEnabled(): Extension[]
```

### getTopicExtensions

Returns the IDs of enabled extensions enabled for a given topic.

```ts
Extensions.getTopicExtensions(topicId: string): Extension[]
```

<ArgsTable
  data={[
    {
      name: 'topicId',
      type: 'string',
      required: true,
      description:
        'The ID of the topic for which to retrieve the extensions.',
    },
  ]}
/>

### addEventListener

Adds extension specific event listeners. Equivalent to calling `addEventListener` directly on `core`, but provides more advanced type definitions.

```ts
Extensions.addEventListener(core: Core, event: ExtensionEvent, callback: EventListenerCallback): void
```

<ArgsTable
  core
  data={[
    {
      name: 'event',
      type: 'ExtensionEvent',
      required: true,
      description:
        'The event type to listen for. See the extension events section below for available events.',
    },
    {
      name: 'callback',
      type: 'EventListenerCallback',
      required: true,
      description:
        'The callback fired when the event occurs. See the extension events section below for the data passed to the callback.',
    },
  ]}
/>
