## Usage

Import the component from the `@minddrop/ui` package.

`'Ctrl'` and `'Alt'` keys will render as `⌘` and `⌥` respectively on Macs.

```jsx
import { KeyboardShortcut } from '@minddrop/ui';

export default () => {
  return <KeyboardShortcut color="contrast" keys={['Ctrl', 'N']} />;
};
```
