## Usage

Import the component from the `@minddrop/ui` package.

```jsx
import {
  Popover,
  PopoverTrigger,
  PopoverContent,
  PopoverClose,
} from '@minddrop/ui';

export default () => (
  <Popover>
    <PopoverTrigger>Popover trigger</PopoverTrigger>

    <PopoverContent>
      <h3>Popover content</h3>
      <p>Are you sure you wanna do this?</p>
      <PopoverClose>Yes</PopoverClose>
    </PopoverContent>
  </Popover>
);
```

## With custom anchor

You can anchor the content to another element if you do not want to use the trigger as the anchor.

```jsx line=11-13
import {
  Popover,
  __PopoverAnchor__,
  PopoverTrigger,
  PopoverContent,
  PopoverClose,
} from '@minddrop/ui';

export default () => (
  <Popover>
    <PopoverAnchor>
      Row as anchor <PopoverTrigger>Trigger</PopoverTrigger>
    </PopoverAnchor>

    <PopoverContent>
      <h3>Popover content</h3>
      <p>Are you sure you wanna do this?</p>
      <PopoverClose>Yes</PopoverClose>
    </PopoverContent>
  </Popover>
);
```

## Styling

### Origin-aware animations

Radix exposes a CSS custom property `--radix-popover-content-transform-origin`. Use it to animate the content from its computed origin based on `side`, `sideOffset`, `align`, `alignOffset` and any collisions.

```css line=14
@keyframes scaleIn {
  from {
    opacity: 0;
    transform: scale(0);
  }

  to {
    opacity: 1;
    transform: scale(1);
  }
}

.popover {
  transform-origin: var(--radix-popover-content-transform-origin);
  animation: scaleIn 0.5s ease-out;
}
```

### Collision-aware animations

Radix exposes `data-side` and `data-align` attributes. Their values will change at runtime to reflect collisions. Use them to create collision and direction-aware animations.

```css line=30-32,34-36
@keyframes slideDown {
  from {
    opacity: 0;
    transform: translateY(-10px);
  }

  to {
    opacity: 1;
    transform: translateY(0);
  }
}

@keyframes slideUp {
  from {
    opacity: 0;
    transform: translateY(0);
  }

  to {
    opacity: 1;
    transform: translateY(-10px);
  }
}

.popover {
  animation-duration: 0.6s;
  animation-timing-function: cubic-bezier(0.16, 1, 0.3, 1);
}

.popover[data-side='bottom'] {
  animation: slideDown;
}

.popover[data-side='top'] {
  animation: slideUp;
}
```
