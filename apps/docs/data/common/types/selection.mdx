# SelectionItem

<Description>
  An interface representing a selected resource.
</Description>

Selected resources (such as drops and topics) are represented using SelectionItem objects.

### SelectionItem

<InterfaceTable
  data={[
    {
      name: 'resource',
      type: 'ResourceReference',
      required: true,
      description: (
        <span>
          A resource reference representing the selected resource.
        </span>
      ),
    },
    {
      name: 'parent',
      type: 'ResourceReference',
      description: (
        <span>
          A resource reference representing the parent resource within
          which the selected resource was selected.
        </span>
      ),
    },
  ]}
/>

```ts
{
  resource: {
    type: 'drops:drop',

}
```
