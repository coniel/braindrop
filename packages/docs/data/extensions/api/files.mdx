# Files

<Description>
  An interface for retrieving and managing content files.
</Description>

## Static methods

### get

Retrieves file references by ID. If provided a signle ID string, returns the file reference or `null` if not found. If provided an array of IDs, returns an array of the corresponding file references.

```ts
Files.get(id: string | string[]): File | File[] | null
```

<ArgsTable
  data={[
    {
      name: 'id',
      type: 'string | string[]',
      required: true,
      description: 'The ID(s) of the file reference(s) to retrieve.',
    },
  ]}
/>

### create

Creates a new file reference from a file and dispatches a `files:create` event. Returns the new file reference.

```ts
Files.create(core: Core, type: string, file: File): FileReference
```

  data={[

### addEventListener

Adds file specific event listeners. Equivalent to calling `addEventListener` directly on `app`, but provides more advanced type definitions.

```ts
Files.addEventListener(core: Core, event: FileEvent, callback: EventListenerCallback): void
```

<ArgsTable
  core
  data={[
    {

    {
      name: 'callback',
      type: 'EventListenerCallback',

  ]}
/>
