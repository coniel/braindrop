# Core

<Description>
  An interface for interacting with MindDrop's core.
</Description>

MindDrop has a lean core mainly responsible for managing event listeners and UI extensions. The core is not aware of any resources such a topics and drops, rather, those are built on top of the core.

### registerResource

Registers a new resource type. Fires a `core:register-resource` event.

You can pass in the resource interface for better type support.

```ts
core.registerResource<Resource>(resource: ResourceConnector<Resource>): void
```

<ArgsTable
  data={[
    {

      description: 'The connector of the resource to register.',
    },
  ]}
/>
