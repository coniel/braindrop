## Usage

Import the component from the `@minddrop/ui` package.

```jsx
import { Toolbar, ToolbarIconButton } from '@minddrop/ui';
import {
  FormatBold,
  FormatItalic,
  FormatStrikethrough,
} from '@minddrop/icons';

export default () => {
  return (
    <Toolbar aria-label="Formatting options">
      <ToolbarIconButton label="Bold">
        <FormatBold />
      </ToolbarIconButton>
      <ToolbarIconButton label="Italic">
        <FormatItalic />
      </ToolbarIconButton>
      <ToolbarIconButton label="Strikethrough">
        <FormatStrikethrough />
      </ToolbarIconButton>
    </Toolbar>
  );
};
```

## Accessibility

Uses [roving tabindex](https://www.w3.org/TR/wai-aria-practices-1.2/examples/radio/radio-1/radio-1.html) to manage focus movement among items.
