## Usage

Import the component from the `@minddrop/ui` package.

```jsx line=7,9
import { __Separator__, Button } from '@minddrop/ui';

export default () => {
  return (
    <div>
      <Button>Blog</Button>
      <Separator orientation="vertical" />
      <Button>Docs</Button>
      <Separator orientation="vertical" />
      <Button>Download</Button>
    </div>
  );
};
```
