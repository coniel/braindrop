## Usage

Menu components are intended for internal use and do not support any interactions by default. To create a dropdown or context menu, use the [`DropdownMenu`](/docs/ui/components/dropdown-menu) or [`ContextMenu`](/docs/ui/components/context-menu) components which build upon these components.

### Composing a Menu

Import the components from the `@minddrop/ui` package.

```jsx
import { Menu, MenuGroup, MenuItem, Icon } from '@minddrop/ui';

export default () => {
  return (
    <Menu>
      <MenuItem
        label="Add title"
        icon={<Icon name="title" />}
        keyboardShortcut={['Ctrl', 'T']}
      />
      <MenuItem
        label="Add note"
        icon={<Icon name="note" />}
        keyboardShortcut={['Ctrl', 'Shift', 'N']}
      />
      <MenuItem
        hasSubmenu
        label="Turn into"
        icon={<Icon name="turn-into" />}
      />
      <MenuSeparator />
      <MenuLabel>Actions</MenuLabel>
      <MenuItem
        label="Copy link"
        icon={<Icon name="link" />}
        keyboardShortcut={['Ctrl', 'Shift', 'C']}
      />
      <MenuItem
        label="Move to"
        icon={<Icon name="arrow-up-right" />}
        keyboardShortcut={['Ctrl', 'Shift', 'M']}
      />
      <MenuItem
        label="Add to"
        icon={<Icon name="inside" />}
        keyboardShortcut={['Ctrl', 'Shift', 'A']}
      />
    </Menu>
  );
};
```

Used to render a label. It won't be focusable using arrow keys.

### MenuSeparator

Used to visually separate items in the context menu. Does not take any props.
