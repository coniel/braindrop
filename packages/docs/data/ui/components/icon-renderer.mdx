## Usage

`IconRenderer` is intended for internal use as a convenient way of rendering the icons of components which accept both `iconName` and `icon` props. It renders an icon from the MindDrop icon set if given an `iconName` or the the value directly if given `icon`. Renders nothing if neither is set.

Import the component from the `@minddrop/ui` package.

```jsx
import React from 'react';
import { IconRenderer, IconProp } from '@minddrop/ui';

interface MyComponentProps {
  icon: React.ReactNode;
  iconName: IconName;
}

export const MyComponent: React.FC<MyComponentProps> = ({
  icon,
  iconName,
}) => {
  return (
    <div>
      <IconRenderer icon={icon} iconName={iconName} />
    </div>
  );
};
```
