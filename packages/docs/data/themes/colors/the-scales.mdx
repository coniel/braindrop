# The Scales

<Description>An overview of MindDrop's color scales.</Description>

## Colors

MindDrop uses six UI color scales and ten content color scales. All colors are designed designed for white foreground text at step `9`. Each color has a light and dark variant.

UI colors are applied to UI elements.
