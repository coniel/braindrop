# Tag

<Description>
  A secondary way of organising content in MindDrop.
</Description>

Tags can be global or topic specific. Global tags are available to all topics. Topic tags are only available within the topic and, optionally, the topic's subtopics.

Drops can have one or more tags applied to them.
